Swinging Jazz pattern group for ESP32 MicroCycles Groovebox R3

Copy the contents of the patterns directory to LittleFS /patterns/.
Use p01.json..p05.json if your firmware expects .json files.
Use p01..p05 if your firmware expects extensionless pattern files.

Chain plan:
p01 -> p02 -> p03 -> p04 -> p02
p05 is the stop/finale pattern.
