# 4. System Settings and Wi-Fi

## Open System Settings

- Use encoder long press.

## Menu Items

Status:

1. SSID
2. IP
3. MAC

Actions:

1. Load Pattern
2. Save Pattern
3. New Pattern
4. Delete Pattern
5. Erase WiFi Credentials
6. Start WiFi Manager
7. Set Theme
8. Rotate Display
9. Encoder Order
10. Restart Groovebox
11. Exit

## Pattern Workflow

- Load Pattern: load a saved pattern from storage.
- Save Pattern: save current pattern to active name.
- New Pattern: save a copy of the current pattern under the next free name.
- Delete Pattern: remove a saved pattern.

Generated pattern names follow this format:

- P001, P002, P003, ...

## Wi-Fi Workflow

- Start WiFi Manager opens a confirmation screen.
- Confirm to launch captive portal mode.
- Connect with your phone/laptop and enter credentials.
- Device reconnects and returns to normal operation.
